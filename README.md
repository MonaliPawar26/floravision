## 🌱 Overview

**Flora Vision** is a modern and responsive frontend web application designed to showcase plant products through an elegant digital interface.

Developed as part of a **frontend development internship task**, this project demonstrates responsive web design, reusable component architecture, and modern frontend development practices using React and TypeScript.

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/MonaliPawar26/Flora-Vision.git
```

### Move into Project Folder

```bash
cd Flora-Vision
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
